Project: Fantelope IOS Mobile Applications

Summary

Client

Fantelope a fan engagement application "Join the herd"

Challenge

Redesign UX and UI for IOS

Outcome

Designed User Experience and User Interface

Deliverables

Wireframes, Low and High-fy mockups of IOS application

Sketch

Adobe Photoshop

Adobe Illustrator

About

Fantelope is a fan engagement application that drives to create a platform that facilitates relationships between B tier college sports.

Situation

They wanted a facelift that integrated modern UI and UX.

Challenge

We had to deal with an extensive wish list of features.

Solution

To meet this desire to scale we doubled down on core features with designs that were ready to integrate features quickly.

Results

We successfully develop a new user interface and experience that was much easier to navigate for users and got them to back to watching the game faster.

Deliverables

  • Wireframes, Low and High fidelity prototypes
